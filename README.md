# 🌋 DrankEiland

Vulkaan-eiland party drinking game voor één iPhone in pass-the-phone modus.

## 📲 Lokaal openen

Pak de zip uit en open `DrankEiland.html` in een browser. Klaar.

## 🚀 Publiceren op GitHub Pages

1. Maak een nieuwe repository op GitHub, bijv. `drankeiland`.
2. Upload **alle** bestanden uit deze map naar de root van de repo:
   - `DrankEiland.html`
   - `manifest.webmanifest`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
   - `README.md` (optioneel)
3. Ga naar **Settings → Pages**.
4. Bij **Source** kies branch `main` en folder `/ (root)`.
5. Klik **Save**. Na ±1 minuut staat het op:
   `https://<jouw-username>.github.io/drankeiland/DrankEiland.html`

## 📱 Installeren als telefoon-app (iPhone)

1. Open de GitHub Pages URL in **Safari** op je iPhone.
2. Tik op het **deel-icoon** (vierkantje met pijl omhoog).
3. Scroll en kies **"Zet op beginscherm"**.
4. DrankEiland verschijnt nu als app-icoon. Tikken opent het fullscreen, zonder browserbalk.
5. Werkt ook **offline** dankzij de service worker.

## 🛠 Wijzigen

Alles zit in één HTML-bestand met embedded CSS + JS — open in je editor en pas aan. Daarna git push en GitHub Pages updatet automatisch.
